Scope Seeker - Seek the scope of your potential

Scope Seeker is a cutting-edge web application designed to help job seekers find their dream job with ease. Whether you are a student looking for your first job or a work professional seeking new opportunities, Scope Seeker has got you covered. We are working on it.

Tech Stack Next.js React.js Chakra UI TypeScript Node.js Express.js MongoDB Docker AWS

Duration: February 2023 - Present

GitHub Live Project
Team Members: Abhijeet Raj, Abhishek Kumar, Harish

Project Purpose and Goal

The purpose of the Scope Seeker project is to revolutionize the job search process for job seekers with specific skills in the tech industry by providing a centralized job search platform that aggregates authentic job postings from various sources. The goal is to create an efficient and effective job finding solution that saves job seekers time and effort while ensuring they have access to relevant and verified job opportunities.

Web Stack Explanation

The chosen web stack, including Next.js, Node.js, MongoDB, Docker, Chakra UI, AWS EC2, GitHub, and Jira Software, was selected for their specific advantages in building a scalable web application. Next.js provides optimized performance and easy configuration. Node.js offers scalability for handling concurrent requests. MongoDB efficiently manages large data volumes. Docker simplifies deployment and management. Chakra UI ensures a user-friendly interface. AWS EC2 provides scalable infrastructure. GitHub facilitates collaborative development, and Jira Software aids in project management.

Problems and Thought Process

I believe if you do all the stuff like database management and writing code then one particularly difficult area was database schema and structure of code. Because of this project's size, I realized how important maintaining an organized structure and database schema would be.

Lessons Learned

I could spend all day describing the lessons that I learned while working on this project, but the most important ones involved my newfound understanding of Node.js, Express.js, and MongoDB. As my first large project using Node.js backend web technology. I learned a lot of lessons regarding code structure, web page routings, and database manipulation.
